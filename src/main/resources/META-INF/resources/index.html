Estudos de Quarkus!

O Quarkus é uma estrutura Java nativa do Kubernetes de pilha completa feita para Java virtual machines (JVMs) e compilação nativa, otimizando o Java especificamente para contêineres e permitindo que ele se torne uma plataforma eficaz para ambientes sem servidor , nuvem e Kubernetes . O Quarkus foi projetado para funcionar com padrões, estruturas e bibliotecas populares de Java, como Eclipse MicroProfile e Spring (demonstrados juntos como parte de uma sessão nesta faixa do Red Hat Summit 2020 ), bem como Apache Kafka, RESTEasy (JAX-RS), Hibernate ORM (JPA), Spring, Infinispan, Camel e muito mais. A solução de injeção de dependência do Quarkus é baseada em CDI (contextos e injeção de dependência) e inclui uma estrutura de extensão para expandir a funcionalidade e configurar, inicializar e integrar uma estrutura em seu aplicativo. Adicionar uma extensão é tão fácil quanto adicionar uma dependência, ou você pode usar as ferramentas do Quarkus. Ele também fornece as informações corretas para GraalVM (uma máquina virtual universal para executar aplicativos escritos em várias linguagens, incluindo Java e JavaScript) para compilação nativa de seu aplicativ
